# 测试想法列表（用于历史洞察功能测试）

以下是10条想法，建议在不同日期输入，以测试历史洞察功能是否能准确识别呼应关系。

## 想法列表（按建议输入顺序）

### 第1天 - 输入以下3条：

1. **工作压力**
   ```
   今天又被老板催项目进度，感觉压力好大。明明已经尽力了，但还是觉得不够好。
   ```

2. **自我怀疑**
   ```
   有时候会想，我是不是真的适合做这份工作？为什么总是觉得自己做得不够好？
   ```

3. **人际关系**
   ```
   和同事聊天时，总是担心说错话。是不是我想太多了？
   ```

---

### 第2天 - 输入以下2条：

4. **工作压力（呼应第1条）**
   ```

   ```

5. **自我认知**
   ```
   今天突然意识到，我可能对自己要求太高了。完美主义有时候反而成了负担。
   ```

### 第3天 - 输入以下2条：

6. **人际关系（呼应第3条）**
   ```
   今天和同事聊得很开心，发现其实不用那么紧张。也许是我之前想太多了。
   ```

7. **情绪变化**
   ```
   感觉最近心情好多了，可能是心态调整过来了。
   ```

---

### 第4天 - 输入以下3条：

8. **工作反思（呼应第1、2、4条）**
   ```
   回顾这段时间，发现我总是在担心工作做得不够好。这种焦虑是从什么时候开始的呢？
   ```

9. **自我成长**
   ```
   今天看了一本书，讲到自我接纳的重要性。也许我应该学会接受不完美的自己。
   ```

10. **未来规划**
    ```
    想换个环境试试，也许新的工作能让我找到更好的状态。但又担心改变带来的不确定性。
    ```

## 预期呼应关系

- **第1条 ↔ 第4条 ↔ 第8条**：都涉及工作压力和自我怀疑
- **第2条 ↔ 第5条 ↔ 第9条**：都涉及自我认知和完美主义
- **第3条 ↔ 第6条**：都涉及人际关系和社交焦虑
- **第8条 ↔ 第10条**：都涉及对工作的反思和未来的不确定性

## 测试步骤

1. 第1天：输入想法 1、2、3
2. 第2天：输入想法 4、5
3. 第3天：输入想法 6、7
4. 第4天：输入想法 8、9、10
5. 在第4天输入完所有想法后，点击"历史洞察"按钮（🔗）
6. 检查是否能准确识别出上述呼应关系

## 注意事项

- 可以在日历中选择不同日期来输入这些想法
- 每条想法输入后，检查控制台是否显示"已打标签"和"已生成embedding"
- 如果某些想法没有关联到，可以调整相似度阈值（当前是0.7）
